# LoRa/LoRaWAN Simulations
LoRa/LoRaWAN Simulations descriptions.

## Simulations Common Parameters 
```bash
FREQ: 915 MHz
BW: 125 kHz
ED TX: 20 dBm
SF, CR: auto, 4/5
Modelo de propagação: Okumura + Nakagami
Período: 7 dias
Repetição de Simulação: 10x

```
